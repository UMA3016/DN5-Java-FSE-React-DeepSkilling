1. angular.json
Configuration file that defines how Angular builds, serves, and tests the project.

2. package.json
Contains project dependencies, scripts, and package information.

3. tsconfig.json
Main TypeScript configuration used across the project.

4. tsconfig.app.json
TypeScript configuration specifically for the Angular application.

5. src/main.ts
Application entry point that bootstraps the root Angular component.

6. src/app/app.config.ts
Contains application-wide configuration such as routing and providers.

7. src/app/app.component.ts
Defines the root component of the Angular application.

8. src/index.html
Main HTML page where the Angular application is loaded.
